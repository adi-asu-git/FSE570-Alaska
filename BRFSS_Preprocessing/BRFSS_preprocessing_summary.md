# BRFSS Data Preprocessing (2020–2024)

The Behavioral Risk Factor Surveillance System (BRFSS) datasets from 2020–2024 were downloaded in SAS transport format (.XPT) and processed using Python. Each annual dataset contains approximately 230k–400k survey responses and more than 250 variables. For the purposes of this project, only variables relevant to obesity and key demographic characteristics were retained. These included the state identifier (_STATE), survey weights (_LLCPWT), body mass index (_BMI5), age group (_AGEG5YR), sex (_SEX), education level (_EDUCAG), income group (_INCOMG / _INCOMG1), and race (_RACE or _RACEGR4 depending on year). Differences in variable naming across survey years were standardized during preprocessing to maintain a consistent schema across the combined dataset.

BMI values in BRFSS are stored as BMI multiplied by 100, so the values were rescaled to obtain the actual BMI. An obesity indicator variable was then created by assigning a value of 1 to respondents with BMI ≥ 30 and 0 otherwise. Observations with invalid BMI codes (e.g., special missing codes such as 7777, 8888, 9999) were removed. Additional filtering steps excluded responses with missing or invalid demographic values based on BRFSS coding conventions for age group, sex, education, and income. These cleaning steps ensured that only valid survey responses with usable demographic information were retained for analysis.

After preprocessing, the cleaned annual datasets were combined into a single multi-year BRFSS dataset covering 2020–2024. The final dataset contains approximately 1.24 million respondents and includes standardized demographic variables, BMI, an obesity indicator, survey weights, and the survey year. This dataset forms the individual-level survey component of the modeling pipeline and will later be used to estimate obesity prevalence across demographic groups, which can then be integrated with county-level demographic distributions from ACS data for small-area estimation.
